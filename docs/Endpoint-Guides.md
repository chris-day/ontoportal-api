# Endpoint Guides

## Purpose
Provide a concise overview of the major OntoPortal/BioPortal API endpoints and link to detailed guides for each resource family.

## Summary
The API is organized around a small set of core resource groups:
- **Ontologies**: list and retrieve ontology metadata and submissions.
- **Classes (Concepts)**: navigate ontology classes and hierarchies.
- **Search**: full-text search across ontologies.
- **Annotator**: annotate free text with ontology concepts.
- **Recommender**: rank ontologies relevant to input text or keywords.
